# LeetCode_Question_Company_Wise
Welcome to the LeetCode Company-Wise Questions repository! This project organizes and maintains a collection of LeetCode problems categorized by companies to help you prepare for technical interviews efficiently.

Features

Comprehensive Question Bank: Contains questions frequently asked in top tech companies.

Organized Structure: Questions are grouped by company for focused preparation.

Updated Regularly: Stay current with newly added LeetCode problems.

Difficulty Levels: Questions categorized as Easy, Medium, or Hard.

Repository Structure

The repository is structured as follows:

.
|-- README.md
|-- Company_Name
    |-- Easy
    |-- Medium
    |-- Hard

For each company, questions are further divided into folders based on difficulty levels.

How to Use

Clone the repository:

git clone https://github.com/ankit1068/LeetCode-Company-Wise-Questions.git

Navigate to the desired company folder:

cd LeetCode-Company-Wise-Questions/Company_Name

Solve questions by referring to the .md or solution files provided.

Contributing

Contributions are welcome! If you'd like to add new questions, improve solutions, or correct errors, follow these steps:

Fork the repository.

Create a new branch:

git checkout -b feature/new-question

Commit your changes:

git commit -m "Add new questions for Company_Name"

Push to the branch:

git push origin feature/new-question

Open a pull request on GitHub.
